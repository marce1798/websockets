# Facturas

>### Descripción
Aplicación de escritorio web para simular el
proceso de registro de facturas.

#### Características
* Crear, editar, actualizar, buscar y borrar productos.
* Calcular precios con iva y totales.
* Almacenar de manera persistentes los datos.
* Representar la facturación en una sección específica.

#### Recuersos usados:
* React.js
* Electron.js
